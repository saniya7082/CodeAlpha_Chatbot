🤖 AlphaBot — Your Friendly ChatBot 



Hello there! Welcome to **AlphaBot** a super cute, lightweight, text-based chatbot built entirely in Python. 



This project was created as part of my CodeAlpha Internship to practice rule-based programming logic, functions, and continuous loop.



✨ Features

&#x20;💬 Continuous Chat:Talk for as long as you want! The bot stays awake until you say goodbye.

&#x20;🧠 Smart Inputs: It understands you even if you use LOUDSPEAKER CAPS or messy spacing. 

⚡Zero Setup: Runs purely inside your console with standard Python—no extra downloads needed.





🛠️ Concepts Practiced

Loops:Using 'while True' to keep the conversation flowing

Functions: Keeping things tidy with modular code blocks.

Control Flow: Using 'if-elif-else' branches to pick the perfect reply.
